# Branding

## Sultan Shawarma — Logo Concept

A logo concept for **Sultan Shawarma**, a shawarma shop that blends Arabian and Turkish cultural elements with a warm, inviting colour palette and a bold skewer icon.

### File

| File | Format | Description |
|------|--------|-------------|
| [`sultan-shawarma-logo.svg`](sultan-shawarma-logo.svg) | SVG | Primary logo (scalable, print & web ready) |

### Design Elements

| Element | Detail |
|---------|--------|
| **Icon** | Bold vertical kebab skewer stacked with grilled meat layers and a green pepper accent |
| **Cultural motifs** | Ottoman-style pointed arch frame, crescent moon, 8-pointed Islamic star, arabesque leaf flourishes, decorative dot ring |
| **Colour palette** | Deep crimson `#8B1A10`, warm gold `#E8A800`, cream `#FFF8EE`, charcoal brown accents |
| **Typography** | Serif (Georgia / Times) — `SULTAN` in gold on a crimson ribbon banner; `SHAWARMA` in deep red below |
| **Tagline** | *Authentic Taste · Timeless Tradition* |

### Possible Variations

- **Urdu typography** — replace Latin lettering with Nastaliq-style Urdu script for a more authentic regional feel.
- **Modern fast-food look** — flat-design version with a sans-serif wordmark and simplified icon for menu boards and packaging.
- **Social media kit** — square and circular cropped versions optimised for profile pictures, story covers, and banner headers.
